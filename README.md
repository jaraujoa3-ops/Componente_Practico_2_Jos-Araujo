## 📋 Descripción del sistema
Aplicación informática diseñada para gestionar el control, registro y seguimiento de la asistencia de estudiantes en instituciones educativas. Permite registrar asistencias diarias, consultar historiales, modificar registros, administrar cursos y materias, así como generar y exportar reportes, sustituyendo el método manual en papel para reducir errores y ahorrar tiempo.

---

## 🎯 Actividad realizada
Se identificaron, redactaron y clasificaron los requerimientos del sistema en tres categorías, cumpliendo con la estructura y requisitos solicitados:
- **Requerimientos funcionales**: Acciones y servicios concretos que el sistema debe realizar
- **Requerimientos no funcionales**: Características de calidad, seguridad y rendimiento
- **Reglas de negocio**: Normas y restricciones institucionales que el sistema debe respetar
